# AgenticAI
This repository contains some practice projects of agentic AI.

## LangChain Agent (this folder)
Minimal Python scaffold for an agent.

### Requirements
- Python 3.12+

### Setup
```bash
python -m venv .venv
.venv\\Scripts\\activate
pip install -U pip
```

### Run
```bash
python main.py
```

### Project Metadata
- Configured via `pyproject.toml`.
